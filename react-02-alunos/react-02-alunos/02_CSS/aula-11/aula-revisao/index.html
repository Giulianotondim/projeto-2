Positions

  1. Static

    Posição padrão dos elementos, não possuem propeidades de posicionamento, para posicioná-los somente com margin

  2. Relative

    Posição relativa à primeira tag static, podendo assim criar um posicionamento relativo à esta, ou seja, permite o uso de propriedades como top, left, rigth e bottom

  3. Absolute

    Posição absoluta em relação ao primeiro elemento relative, onde as propriedades top, left, rigth e bottom definem o afastamento efetivo das bordas decritas

  4. Fixed

    Posição fixa em relação à primeira tag static, onde as propriedades top, left, rigth e bottom definem o afastamento efetivo das bordas decritas independente do scroll do mouse
